Homzen Theme Icon Reference

Complete list of all available icons using the IcoMoon font system

How to Use Icons

1. HTML Class Method (Most Common)

<i class="icon icon-home"></i>
<span class="icon icon-bed"></span>

2. Laravel Blade Component (New Method)

<x-core::icon name="ti ti-heart" />

3. Custom Styling

<i class="icon icon-star" style="color: gold; font-size: 24px;"></i>
